Akash's Cafe Website
Akash's Cafe is a full-stack cafe ordering and reservation website built with React, Vite, Node.js, Express.js, and SQLite. The project includes a modern cafe interface where users can browse menu items, add food to cart, apply coupons, register/login, place orders, submit reviews, and reserve tables.


Features:

Responsive cafe website UI
Home, Menu, Cart, Reserve, Reviews, and Contact sections
Live menu listing with categories and search
Add to cart, update quantity, and remove items
Coupon discount system
User registration and login
Token-based session handling
Order placement and order history
Table reservation system
Customer review system
Backend API using Express.js
Local database storage using SQLite


Technologies Used:

HTML: Provides the base structure for the website.
CSS: Handles styling, layout, colors, responsive design, and visual appearance.
JavaScript: Adds interactivity and application logic.
React.js: Builds the frontend user interface using reusable components and state.
Vite: Runs the frontend development server and builds the React app.
Node.js: Runs JavaScript on the server side.
Express.js: Creates backend API routes for menu, users, orders, reviews, and reservations.
SQLite: Stores project data locally in cafe.db.
npm: Installs dependencies and runs project scripts.
Concurrently: Runs frontend and backend together during development.

Project Explanation:

This project is a full-stack cafe website. The frontend is developed using React.js and styled with CSS. The backend is developed using Node.js and Express.js. The frontend sends API requests to the backend for actions such as login, registration, orders, reservations, and reviews. The backend stores the data in a SQLite database file named cafe.db.
